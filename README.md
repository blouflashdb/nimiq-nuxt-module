## Quick Setup

Install the module to your Nuxt application with one command:

```bash
npx nuxi module add nimiq-nuxt-module
```

That's it! You can now use nimiq-nuxt-module in your Nuxt app

  <summary>Local development</summary>
  
